Информация о сайте

На этом сайте вы можете создавать онлайн турниры по математическому домино и участвовать в них.

Пользователи со статусом учителя могут создавать турниры, просматривать и добавлять задачи.

Пользователи со статусом ученика могут участвовать в турнирах.

При регистрации всем пользователям по умолчанию присваивается статус ученика. Для получения статуса учителя необходимо связаться с администрацией сайта.

Правила математического домино

В игре 28 задач, их номера соответствуют номерам костяшек домино. На руках у игрока одновременно может находиться только одна задача, которую он может решать и сдавать, условий остальных задач он не видит.

Первая задача выдается игрокам автоматически, далее после каждой сдачи ответа необходимо взять из банка новую задачу. В банке доступны для выбора только задачи, которые в данный момент не находятся на руках у игроков.

По каждой задаче, кроме задачи 0-0, есть две попытки. Если задача правильно сдана с первой попытки, игрок получает за нее количество баллов, равное сумме чисел на доминошке. Если задача не верно сдана с первой попытки, игрок получает по ней желтую карточку и может взять задачу повторно. При правильном ответе со второй попытки игрок получает количество баллов, равное большему из чисел на доминошке. Если же игрок повторно сдает задачу неправильно, он получает красную корточку и у него отнимается число баллов, равное меньшему числу на доминошке.

По задаче 0-0 есть только одна попытке. При правильном ответе игрок получает 10 баллов, про неправильном 0 по задаче и красную карточкую.
